How to create a token?

Creating a GitHub Personal Access Token:

  1. Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens
  2. Click "Generate new token (classic)"
  3. Give it a descriptive name like "MTG Deck Submission"
  4. Set expiration as needed
  5. Select the "repo" scope (full control of private repositories)
  6. Click "Generate token" and copy the generated token

Note: Keep your token secure and never share it publicly.
